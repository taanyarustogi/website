No Touch Disinfectant Wipes Dispenser

In grade 12, during covid since all schooling was moved online, I along with other peers from all over the world competed in an online hackathon, the theme of which was to create a project that helps lessen the spread of Covid-19. The project that I decided on creating is a disinfectant wipes dispenser that is touch-free and uses an ultrasonic sensor along with an arduino coded in python to sense the users hand waving which moves a motor. The motor is connected to a rod which is connected to a roll of disinfectant wipes that unrolls when the motor moves. Whilst completing this project, I learned not only how to use hardware components such as breadboards and arduinos but also how to code in python as that is what was required when working with arduinos.

Child Tracker - 3D printed

In grade 9 and 10 as part of my IGCSE certification I completed a Design and Technology: Systems and Controls course where we created a couple of projects whilst learning about structures, electronics and mechanism. We made and tested a bridge, synthesizer and an automata. As a part of my final project I made a two piece child tracker which send signals to each other and see the strength of the signal. When the signal's strength is weaker than a certain amount, the parent part of the child tracker shows a cross indicating that the child is far from the parents where otherwise the tracker shows a check mark. The child part of the child tracker starts emmiting a noise when the signal's strength is less than a certain amount making it easier for the parent to find the child. For this project, I used two MicroBits and 3D printed the bodies of the child trackers after making them on fusion.

Reversi Lab

In first year of university semester two we did a course learning C programming which was our first programming course. At the end of the semester our final project was to code a computer opponent for the game reversi or othello, attached is the github link for the code used. In this code, the user chooses the dimensions for a square othello board and a text-based image of the board is shown. The user then choses which color the computer plays and the game starts with the computer asking the user to input a letter coordinate and the computer printing the board. The computer then plays its move and prints the board with its move shown. This continues until no more moves can be made and the computer determines who wins. If the user makes an invalid move, the computer prints out 'invalid move' and the game ends there.

Microcontroller unit for Radio - design course

In second year second semester we had a hardware design course, ECE295, where each class of 6/7 groups of 3 students was given a part of a radio to design and at the end all teams would come together to make a complete radio. My group's part was a microcontroller, ATMega34A, that took an two inputs from the user, a frequency using a potentiometer and the mode (transmitting or receiving) through a toggle switch. The MCU then used I2C connections to send the frequency to a PLL that transmits two sinusoidal waves 90 degrees out of phase to the other subsystem mixers and a signal sent to the relay in the switch to switch between the modes. We started the project from scratch, created the schematic, the PCBs, coded the MCU in C, debugged and learned alot along the way.

C game played on DE1-SoC Computer - ARM Language

This game was made as a final project for ECE243 which is an ARM assembly course, however this game is coded in C and played on a FPGA board. The keyboard arrows are used to control a white colored user car, the goal of the game is to not crash onto another cars which are coming towards the user at varying speeds. You use the 'esc' key to start the game and can choose levels using the numbers 1-3 when the game has started. The spacebar can also be used to pause the game and enter to unpause the game. The seven-segment display shows the number of cars that the user has doged, the 'score'. When the user bumps into a car the game ends and a 'game over' screen is displayed, the user can press the 'esc' key again to restart the game.
